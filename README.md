
# Local RAG Assistant

A private, local Retrieval-Augmented Generation (RAG) system built with LlamaIndex, Ollama, and Streamlit.

## 🛠️ Tech Stack
- **Framework:** LlamaIndex
- **LLM:** Ollama (Llama 3/Mistral)
- **Vector Store:** ChromaDB
- **Embeddings:** HuggingFace
- **UI:** Streamlit
